# JulesApp SDK for Google Apps Script

A powerful client library for accessing the Jules API directly from Google Apps Script. Automate your workflows, manage sessions, and integrate Jules intelligence into Google Docs, Sheets, and more.

## Installation

### 1. Add the Library
To use this SDK in your Google Apps Script project:

1.  Open your project in the Apps Script editor.
2.  Click **Libraries (+)** in the left sidebar.
3.  Paste the Script ID:
    `1Qnm6YgwvY2gmtnPQuC8mtH8I9rbIQdC0MTgO466P4cn1Jk-CE52YWMVg`
4.  Click **Look up**.
5.  Select the latest version and click **Add**.

### 2. Configure API Key
For security, store your API key in the Script Properties.

1.  Click the **Project Settings** (Gear Icon ⚙️) on the left sidebar.
2.  Scroll down to **Script Properties**.
3.  Click **Add script property**.
    *   **Property**: `JULES_API_KEY`
    *   **Value**: `YOUR_ACTUAL_API_KEY`
4.  Click **Save script properties**.

## Quick Start

Here is the minimal code needed to start a session with Jules.

```javascript
function startJulesSession() {
  // 1. Retrieve your API key securely
  var scriptProperties = PropertiesService.getScriptProperties();
  var apiKey = scriptProperties.getProperty('JULES_API_KEY');

  // 2. Initialize the SDK
  JulesApp.setApiKey(apiKey);

  // 3. Create a Session
  var session = JulesApp.createSession({
    prompt: "Review the PR guidelines",
    sourceContext: {
      // Replace with your actual source resource name
      source: "sources/github/your-org/your-repo"
    }
  });

  Logger.log("Session Created: " + session.id);
}
```

## Examples

### Create a Task from Google Docs Selection

This example demonstrates how to build a productivity tool that takes highlighted text from a Google Doc and sends it to Jules as a new task.

```javascript
function createTaskFromHighlight() {
  // 1. Get the user's selection
  var selection = DocumentApp.getActiveDocument().getSelection();

  if (!selection) {
    DocumentApp.getUi().alert('Please highlight the text you want to turn into a task.');
    return;
  }

  // 2. Extract text from the selection
  var text = "";
  var elements = selection.getRangeElements();
  for (var i = 0; i < elements.length; i++) {
    var element = elements[i].getElement();
    // Simple text extraction
    if (element.getType() === DocumentApp.ElementType.TEXT) {
      text += element.getText() + " ";
    }
  }

  // 3. Initialize Jules
  var props = PropertiesService.getScriptProperties();
  JulesApp.setApiKey(props.getProperty('JULES_API_KEY'));

  try {
    // 4. Create the session with the selected text
    var session = JulesApp.createSession({
      prompt: "Create a feature plan based on these requirements: " + text,
      title: "Doc Requirement Task",
      sourceContext: {
        source: "sources/github/your-org/your-repo"
      }
    });

    DocumentApp.getUi().alert('Success! Jules is working on it.\nSession ID: ' + session.id);
  } catch (e) {
    DocumentApp.getUi().alert('Error creating task: ' + e.message);
  }
}
```

## API Reference

### Configuration
*   `setApiKey(key)`: Sets the API key for all subsequent requests.

### Sessions
*   `createSession(config)`: Starts a new session.
*   `getSession(idOrName)`: Retrieves details of an existing session.
*   `listSessions(pageSize, pageToken)`: Lists recent sessions.

### Sources
*   `listSources(pageSize, pageToken)`: Lists connected sources (e.g., GitHub repositories).
*   `getSource(idOrName)`: Gets details about a specific source.

### Activities & Actions
*   `listSessionActivities(sessionId)`: specific activities for a session.
*   `sendMessage(sessionId, message)`: Sends a follow-up message to the agent.
*   `approvePlan(sessionId)`: Approves the plan generated by the agent.
